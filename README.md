# Instalação
Clone o repositório:
- git clone https://github.com/MariaEGSilva25/Prensadinho-Grill-Backend
- cd Prensadinho-Grill-Backend

## Execute a Aplicação

- docker-compose up -d
- ./mvnw spring-boot:run
